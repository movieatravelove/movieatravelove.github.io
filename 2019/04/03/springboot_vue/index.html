二、每次请求session 不一致问题

  • 后端 Controller

    登录接口返回sessionId

  • 前端 vue

    修改前端axios请求，在axios的全局配置

    这句话的意思是每次请求都带上cookie,这样每次取请求后端,就不会被认为是不同用户导致session丢失.哦了!
